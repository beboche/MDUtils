# MDUtils
Utilities for MobiDetails


## Installation


requires python 3

### In regular environment

```bash

git clone https://github.com/beboche/MDUtils.git

```

then

```bash

cd MDUtils
pip install -r requirements.txt

```
Done!

### In a virtual environment


```h

git clone https://github.com/beboche/MDUtils.git
cd MDUtils
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt

```
Done!
